# Retailer Data Analysis & Power BI Dashboard 📊🛒

An end-to-end data analysis and business intelligence project focused on exploring retail transaction data, understanding customer buying patterns, optimizing inventory metrics, and building an interactive dashboard to drive revenue growth.

## 🚀 Project Overview
This repository contains a comprehensive data analysis workflow conducted on a retailer dataset alongside an interactive **Power BI Dashboard**. The primary goal is to transform raw transactional logs into actionable business intelligence regarding sales performance, product trends, and consumer behavior.

## 📊 Power BI Dashboard Features
In addition to the Jupyter Notebook analysis, a dynamic Power BI report was developed to provide stakeholders with real-time, executive-level insights:
* **Executive Summary:** KPI cards tracking Total Revenue, Total Orders, Average Order Value (AOV), and profit margins.
* **Sales Performance Trends:** Interactive time-series charts (Year-over-Year and Month-over-Month growth).
* **Customer Demographics & Behavior:** Slicers to filter sales by region, store location, and customer segments.
* **Inventory & Product Analysis:** Top-selling products matrix and return rate analysis to monitor stock health.

## 🧠 Key Analysis Objectives (Python)
* **Sales Performance:** Identify top-performing products, categories, and seasonal revenue trends.
* **Customer Behavior:** Segment customers based on purchasing frequency, monetary value, and recency (RFM Analysis).
* **Geographical Insights:** Analyze sales distribution across different regions/stores.
* **Operational Efficiency:** Detect patterns in return rates, discount impacts, and peak order times.
